My Projects

BruinMeet

I'm currently one of the back-end developers for BruinMeet, which is a UCLA dating app. We use Node.js on the back-end, and I've worked on features such as custom profile pictures and mobile notifications.

I'm planning on learning how to implement machine learning algorithms into our matchmaking process.

2048 Dueling Deep Q-Network

For my final project in MATH 156 (Machine Learning), I worked with a group on a dueling DQN for the game 2048. The network was implemented in Python with TensorFlow.

The network does pretty well—it scores 12000 points on average, and it hits the 2048 tile roughly 5% of the time after sufficient training.
